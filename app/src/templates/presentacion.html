Introduccion

Angular un framework que nacio en el 2009, la empresa que se le da soporte es Google. Tiene la capacidad de brindarnos la posibilidad de extender las funcionalidades del html, es un framework que tiene como objetivo la modularidad de sus componentes. para brindarnos ciertas ventajas (encapsulacion de logica, mayor rapidez a la hora de ejecutar test, inyecti√≥n de dependencias).

Es orientado a las aplicacion Single Page App(SPA).

Expresiones

Las expresiones en angualar se usan en la parte del "view" como una forma sencilla de mostrar datos, teniendo la habilidad realizar operaciones computadas por ejemplo sumar,restar entre otras.

Las expresiones en angular a diferencia de las de javaScript, se manejan dentro de un ambito o "scope" que sera el contexto donde viviran. No se puden utilizar bucles, excepciones o condicionales dentro de las expreciones. Se pueden utilizar filtros en conjunto con las expresiones.
